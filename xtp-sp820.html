TPMS PROGRAMMING XTP-SP820

อุปกรณ์อ่านเซ็นเซอร์วัดความดันลมยาง

ขนาดจอ : หน้าจอ หน้าจอ TFT สี ขนาดประมาณ 2.8 นิ้ว (320x240 พิกเซล)

หน่วยความจำ : RAM 64 MB

แบตเตอรี่อุปกรณ์อ่านเซ็นเซอร์วัดความดันลมยาง : Li-polymer 3.7 โวลต์ ความจุ 3000 mAh

รองรับเมนูภาษา : 16 ภาษา (รวมภาษาไทย)

คุณสมบัติ

TPMS PROGRAMMING XTP SP820

  • อ่านข้อมูลเซนเซอร์: ID, แรงดันลม, อุณหภูมิ, แบตเตอรี่
  • วินิจฉัยระบบ TPMS: อ่านโค้ด, ล้างโค้ด, ตรวจสอบสถานะระบบ
  • โปรแกรม / เขียน ID ให้เซนเซอร์: รองรับ 4 วิธี (อัตโนมัติ, กรอกด้วยตนเอง, โคลน, OBD)
  • การกระตุ้นเซนเซอร์ : สามารถกระตุ้น/ปลุกให้เซนเซอร์ส่งข้อมูลได้

UNIVERSAL SENSOR TPS 800-1

  • ความถี่สัญญาณเซ็นเซอร์วัดลมยาง : มีความแม่นยำสูง แสดงสถานะได้อย่างรวดเร็ว
  • วัสดุ : โลหะ เพื่อลดการรั่วซึม รองรับอุณหภูมิการใช้งานสูงและกันน้ำได้เป็นอย่างดี
  • รองรับรุ่นรถ : ครอบคลุมรถยนต์กว่า 98%
  • แบตเตอรี่ : อายุยาวนานถึง3ปี หมดปัญหาเปลี่ยนแบตเตอรี่บ่อย
จุดเด่น

TPMS PROGRAMMING XTP SP820

  • ฟังก์ชันครบในเครื่องเดียว: อ่าน, วินิจฉัย, โปรแกรม, และฟังก์ชัน OBD II ในอุปกรณ์เดียว
  • รองรับการอัปเดตซอฟต์แวร์ : ผ่าน USB และ WI-FI สามารถเพิ่มรุ่นรถใหม่ได้ตลอดเวลา
  • ออกแบบสำหรับใช้งานในอู่ศูนย์บริการยางรถยนต์ : ง่ายต่อการใช้งาน ประหยัดค่าใช้จ่ายในการเก็บสต็อกเซนเซอร์ที่หลากหลาย

UNIVERSAL SENSOR TPS 800-1

  • ความทนทาน : วัสดุโลหะ เพื่อลดการรั่วซึม รองรับอุณหภูมิการใช้งานสูงและกันน้ำได้เป็นอย่างดี
  • ใช้แทนเซ็นเซอร์ OE : เพื่อแสดงผลกับระบบเดิมได้ มาตรฐานเดียวกับเซนเซอร์จากโรงงาน
